# IH-TaskManager

## Project Description
IH-TaskManager is a web-based application that allows users to manage tasks efficiently. Users can create, edit, and track tasks, view task progress through charts, and monitor deadlines. The website also supports multiple users with different roles.


## Features
1. **User Authentication:** Login system supporting multiple users with different roles which is admin and students
2. **Dashboard:** Overview total tasks, due today tasks, completed tasks, and incomplete tasks that includes with progress bars and donut chart.
3. **Task Management:** Create, edit, delete, and filter tasks by status such as in progress, not started and completed or due date.
4. **User Management:** Admin can view all registered users with details like avatar, name, username, email, password, and role.
5. **Responsive Design:** Works on both desktop and mobile devices.
6. **Bootstrap Toast Messages:** Displays notifications for actions like task creation, update, deletion, or login errors. Toasts appear briefly at the top right corner and automatically disappear to ensure providing a smooth user experience.

## Frameworks / Libraries Used
1. **Bootstrap 5** – for responsive layout, styling, and toast notifications.  
2. **Font Awesome** – for icons.  
3. **Chart.js** – for displaying task statistics as charts.  
4. **JavaScript** – for task CRUD operations, user management, dynamic content, and triggering toast messages.
